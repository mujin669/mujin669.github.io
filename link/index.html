字段解释

  1. width代表边框的大小，0 代表不显示。默认为（0px，也就是不显示）

  2. color 代表色的颜色，也是鼠标悬停后的颜色、字体颜色。默认为淡蓝色（#49b1f5）

可以使用的颜色格式：rgb、rgba、hsl、hsla、16 进制颜色以及关键字颜色等

也可以设置渐变色，但是渐变色会导致边框颜色及字体颜色为默认色无法更改。

  1. style 边框的形状也就是 border-style 的取值，默认为实线（solid）

  2. time动画时长设定，默认为 0s，也就是不添加动画。

  3. rotate填写角度代表鼠标悬停时头像旋转的角度，默认为 0deg，不旋转。

  4. custom此字段留下来的目的时为了后期添加更多的特效。默认值 link_custom

  5. link_custom 为呼吸灯效果

flash 为边框闪现效果

渐变配色参考网址： Gradient Coolhue Uigradients
